## 0.1.0

* Initial release of Flutter Urovo RFID plugin
* Added support for Urovo RFID hardware integration
* Implemented core functionality:
  - Power on/off and connect to RFID reader
  - Start/stop continuous inventory scanning
  - Perform single inventory scans
  - Retrieve firmware version
  - Stream tag events (EPC, TID, RSSI)
* Added example application demonstrating plugin usage
* Added comprehensive documentation in README.md
